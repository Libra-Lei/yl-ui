# Upload 上传组件

提示

简易的上传组件，可点击或者拖拽上传

# 基础用法

upload

拖拽文件到此处或点击选择

    <template>
  <div>
    <yl-upload-mini v-model="files" />
  </div>
</template>

<script>
export default {
  name: 'Upload',
  data() {
    return {
      files: []
    }
  }
}
</script>
